# 🌗 Theme Toggle (Day 2 / 21 Challenge)

A simple theme toggle project built using JavaScript. Users can switch between dark and light modes to improve readability and user experience.

---

## 🎯 Goal

To allow users to change the theme of a webpage for better clarity and comfort.

---

## ✨ Features

- 🌙 Toggle between dark and light mode  
- 🎨 Instant UI color change  
- ⚡ Smooth user interaction  

---

## 🔄 Project Flow

1. User clicks on toggle button  
2. JavaScript checks current theme  
3. If dark → switch to light  
4. If light → switch to dark  
5. UI updates accordingly  

---

## 🧠 Logic Used

- `onclick` event  
- Conditional statements (`if-else`)  
- DOM manipulation  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
